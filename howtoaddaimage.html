How to add a image

The internet would be rather dull and boring without images so to liven up your HTML page, we'll show you how to add a image to your HTML page. 

First, make sure that you have an image you can use. It doesn't have to be any particular size, just an image. We're going to use a image on our desktop, but you can use one from Google Images, for example. Once you've found an image you like, just right click and select Copy Image Address (on Chrome). 

Make sure to:

  1. COPY THE FILE PATH - If you're using a internally stored image, make sure to get the correct file path. If the image is stored where the HTML file is, then all you have to enter is the file name (for example, image.png) 
  2. FIND THE FILE EXTENSION - If you're using a internally stored image, check the file extension (such as .jpg, .png) as you'll need to know this when linking the image. 

All you have to do is add this piece of code to your HTML page (make sure to change the filename and file extension if necessary)

<img src="image.png" </img>
